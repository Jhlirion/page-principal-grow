# Carpeta de imágenes — NexoIA

Cuando tengas fotos propias, súbelas aquí con estos nombres exactos
y luego actualiza las URLs en css/styles.css (sección "IMÁGENES").

| Archivo       | Dónde aparece                  | Tamaño recomendado |
|---------------|--------------------------------|--------------------|
| hero.jpg      | Hero principal (derecha)       | 1200 × 1600 px     |
| equipo.jpg    | Sección Nosotros               | 900  × 1100 px     |
| banner1.jpg   | Banner entre Servicios/Proceso | 1600 × 600  px     |
| banner2.jpg   | Banner antes del FAQ           | 1600 × 600  px     |
| cta.jpg       | Fondo sección CTA final        | 1600 × 700  px     |
| og-image.jpg  | Preview en WhatsApp/redes      | 1200 × 630  px     |

## Cómo cambiar una imagen

1. Sube tu foto aquí con el nombre exacto de la tabla
2. Abre `css/styles.css`
3. Busca la sección `/* ── IMÁGENES ──`
4. Cambia la URL de Unsplash por:  images/nombre-foto.jpg

Ejemplo:
  --img-hero: url('images/hero.jpg');
